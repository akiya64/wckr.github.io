# How to Develop on Local Machine

## Requires

- Ruby
- Jekyll
- Compass

## Getting Started

### Jekyll

Serve the site locally via Jekyll

```bash
$ jekyll s -w
```

### Compass

Open another terminal window to run Compass

```bash
$ compass w
```

### Visit the local site

The default URL will be http://localhost:4000
